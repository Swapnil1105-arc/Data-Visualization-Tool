> **Transform raw CSV data into meaningful visual insights with just a few inputs.**
>
> A lightweight, interactive command-line application that enables users to explore datasets through powerful visualizations using **Pandas**, **Matplotlib**, and **Seaborn**.

---

# ✨ Features

✅ Load any CSV dataset instantly

✅ Interactive menu-driven interface

✅ Display complete dataset or preview records

✅ Generate multiple types of visualizations

✅ Automatic Correlation Heatmap

✅ Error handling for missing files and invalid inputs

✅ Beginner-friendly and easy to extend

---

# 📸 Supported Visualizations

| Visualization          | Purpose                                     |
| ---------------------- | ------------------------------------------- |
| 📈 Line Chart          | Analyze trends over time                    |
| 📊 Bar Chart           | Compare categories                          |
| 🔵 Scatter Plot        | Identify relationships between variables    |
| 📉 Histogram           | Understand data distribution                |
| 📦 Box Plot            | Detect outliers and spread                  |
| 🔥 Correlation Heatmap | Find relationships among numerical features |

# 🚀 Installation

## 1. Clone the Repository

```bash
git clone https://github.com/Swapnil1105-arc/Data-Visualization-Tool.git
```

```bash
cd Data-Visualization-Tool
```

---

## 2. Install Dependencies

```bash
pip install pandas matplotlib seaborn
```

or

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Project

```bash
python "Data Visualization Tool.py"
```

---

# 📋 How It Works

### Step 1

Run the application.

### Step 2

Enter the path of your CSV file.

Example:

```text
Enter CSV file name:
sample_dataset.csv
```

### Step 3

The dataset is loaded successfully.

### Step 4

Choose any visualization from the interactive menu.

```text
========== MENU ==========

1. Display Dataset
2. Line Chart
3. Bar Chart
4. Scatter Plot
5. Histogram
6. Box Plot
7. Correlation Heatmap
8. Exit
```

### Step 5

Enter the required column names and enjoy the visualization.
